## üìú Game Rules

Each player (you and the computer) selects one of the following options:

- üêç **Snake (`s`)** drinks the water ‚Üí Snake defeats Water
- üíß **Water (`w`)** damages the gun ‚Üí Water defeats Gun
- üî´ **Gun (`g`)** kills the snake ‚Üí Gun defeats Snake

> If both choices are the same, the round is declared a **draw**.
